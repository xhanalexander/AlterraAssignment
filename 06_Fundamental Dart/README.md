## Summary

### Dart

Dart merupakan bahasa pemrograman dengan tujuan yang dirancang untuk membangun aplikasi cepat di berbagai platform dan dikembangkan oleh perusahaan google yang pertama kali muncul pada tahun 2011  untuk menjadi alternatif dari bahasa pemrograman javascript. type safe menjamin konsistensi data tidak ambigu dan null safety, yang memberikan data null atau data kosong dan juga didukung banyak library.
